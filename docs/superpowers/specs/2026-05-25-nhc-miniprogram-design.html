国家卫健委政策文件微信小程序 — 产品设计文档

项目概述

微信小程序，将国家卫健委每日发布的政策文件分类整理，支持按类别检索、关键词搜索、收藏和推送提醒。先做个人体验版，验证后正式上架。

目标用户

画像 典型角色 核心场景 核心诉求
医疗从业者 医院科室主任、医务科人员 每天扫一眼有无新文件 及时性 > 全面性
合规/法务人员 药企合规官、律所医疗团队 接到任务时集中检索 全面性 > 及时性

核心痛点：手动刷官网效率低、靠微信群转发易遗漏、搜索功能烂。

MVP 功能范围

优先级 功能 说明
P0 最新文件列表 首页时间倒序展示，上滑加载更多
P0 分类浏览 一级按业务领域（标签）、二级按发文类型（筛选栏）
P0 关键词搜索 搜索标题 + 发文机构，搜索结果高亮关键词
P0 文件详情页 元信息 + AI 关键内容提取 + 查看原文 + 附件
P0 收藏/书签 详情页右上角收藏，首页右上角入口查看收藏列表
P0 新文件推送 微信订阅消息，每日 9:00 推送昨日新增文件摘要
P0 最近更新时间 首页显示数据同步时间
P0 全文阅读 关键内容提取（后端 AI 生成要点摘要）

本期不做：多条件组合筛选、历史归档（2026 年以前）。

视觉风格

  • 主题：政务蓝 #1a6fb5，白底，专业稳重
  • 风格关键词：权威、清晰、克制

信息架构

🏠 首页
 ├─ 🔍 搜索栏 → ④ 搜索页 → ③ 详情页
 ├─ ⭐ 收藏入口 → ⑤ 收藏页 → ③ 详情页
 ├─ 📂 领域标签 → ② 分类列表页 → ③ 详情页
 ├─ 📄 文件卡片 → ③ 详情页
 └─ 查看更多 → ② 分类列表页 → ③ 详情页

全局规则：所有页面顶部均有 ← 返回按钮，返回上一级页面。

页面设计

① 首页

  • 顶部：搜索栏（圆角输入框，placeholder "搜索标题、发文机构..."）+ 右上角 ⭐ 收藏入口
  • 更新时间：「最近更新：YYYY-MM-DD HH:MM」，灰色小字
  • 领域标签：横向排列的圆角标签，"全部"为选中态（蓝底白字），其余为未选中（浅蓝底蓝字）
  • 类型筛选栏：二级分类（全部类型 / 规范性文件 / 政策解读 / 通知公告），选中态下划线
  • 文件列表：卡片式，每项展示标题（加粗）、文号 + 日期。上滑加载更多
  • 底部：「查看更多 →」按钮，跳转当前筛选条件下的分类列表页

② 分类列表页

  • 顶部：← 返回 + 当前分类标题（如"医政管理 · 规范性文件"）+ 文件总数
  • 列表：同首页文件卡片，上滑分页加载

③ 文件详情页

  • 顶部：← 返回，右上角 ★ 收藏按钮（已收藏实心蓝星，未收藏空心灰星）
  • 标题：加粗，字号较大
  • 元信息：文号、发文日期、发文机构、所属分类
  • 关键内容：📌 区域，❶❷❸ 要点列表（后端 AI 生成），灰底卡片
  • 查看原文：蓝色实心按钮，突出显示，跳转 NHC 原文链接
  • 附件：📎 列表，浅蓝底卡片，点击下载/预览

④ 搜索页

  • 顶部：← 返回 + 搜索输入框（自动聚焦）
  • 结果数：「找到 N 条结果」
  • 结果列表：同文件卡片，关键词高亮（<mark> 标签），上滑分页
  • 空状态：📭 图标 + "暂无结果" + "换个关键词试试"

⑤ 收藏页

  • 顶部：← 返回 + "⭐ 我的收藏" + 收藏总数
  • 列表：文件卡片 + 右侧实心 ★（点击取消收藏）
  • 空状态：📭 图标 + "暂无收藏"

⑥ 推送消息

  • 渠道：微信小程序订阅消息
  • 触发：每日 9:00 推送
  • 内容：昨日新增文件数量 + 标题列表（最多 3 条）
  • 点击：跳转小程序首页
  • 首次引导：弹窗引导用户订阅，可选"去开启"或"暂不需要"

交互状态

状态 表现
加载中 骨架屏（灰色占位块）+ "加载中..."
下拉刷新 首页下拉触发，显示刷新动画
空数据 图标 + 提示文案
网络异常 🌐 图标 + "网络异常，请检查网络后重试" + "重新加载"按钮
操作反馈 Toast 轻提示，2 秒消失（如"⭐ 已加入收藏"）
首次打开 订阅消息引导弹窗

分类体系

一级领域：全部 / 医政管理 / 疾控 / 妇幼健康 / 基层卫生 / 药政 / 医管 / ...

二级类型：全部类型 / 规范性文件 / 政策解读 / 通知公告 / ...

两种维度叠加筛选，一级选领域、二级选类型，列表联动。
